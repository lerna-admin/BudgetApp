# budgetapp - Venture Profile

- venture_id: budgetapp
- name: BudgetApp
- owner: xanadu
- stage: build

## Problem

Usuarios no tienen un sistema simple para controlar gastos, deudas y cuentas en un solo flujo diario, lo que genera baja claridad financiera y malas decisiones.

## Target user

- persona_1: profesionales independientes con ingresos variables
- persona_2: parejas o roomies que reparten gastos
- persona_3: usuarios que quieren controlar deudas y presupuesto mensual

## Value proposition

BudgetApp centraliza gastos, deudas y presupuesto con interfaz simple y automatizaciones para seguimiento diario con baja friccion.

## 90-day goals

1. Lanzar presencia online y onboarding con propuesta clara.
2. Activar ciclo de mejora continua de producto basado en datos reales.
3. Automatizar reportes operativos para reducir trabajo manual.

## KPIs

- north_star: weekly_active_budgeters
- acquisition: new_signups_per_week
- activation: users_with_first_5_expenses_in_48h
- retention: week_4_return_rate
- revenue: conversion_to_paid_or_premium_plan
