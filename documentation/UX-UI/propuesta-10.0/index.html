Propuesta 10.0 · Pantallas finales (HTML + CSS + JS)

Disenos finales para implementacion (web + mobile)

Esta propuesta separa cada pantalla en su propio directorio con un index.html. El documento principal carga cada pantalla dentro de un iframe para verla como se veria en el navegador. Todas las pantallas internas son responsive: la vista cambia segun el ancho del viewport (Desktop/Mobile).

Recomendado: abrir por http://localhost (no file://) para evitar restricciones de iframes. Ej: python3 -m http.server 8000 y abrir /documentation/UX-UI/propuesta-10.0/index.html.
